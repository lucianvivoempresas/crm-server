🔧 COMO FUNCIONA TECNICAMENTE:

  1. Usuário digita CNPJ em campo input
  2. Listener aguarda "blur" (sair do campo)
  3. setupCNPJListener() ativa
  4. fetchCNPJData() executa
  5. Requisição HTTP para ReceitaWS:
     GET https://www.receitaws.com.br/v1/cnpj/{CNPJ}
  6. API retorna JSON com dados da empresa
  7. loadCNPJDataInForm() preenche os campos
  8. Usuário valida e salva
